# TC-003: Verify selected price sort option remains visible after selection

## Test Case ID
TC-003

## Title
Verify the selected price sort option remains visible in the Sort By dropdown after selection

## Related User Story
Sort product listings by price

## Preconditions
- User is on the Product Listing Page (PLP)
- Sort By dropdown is available

## Test Steps
1. Navigate to the Product Listing Page
2. Open the **Sort By** dropdown
3. Select **Price: Low to High**
4. Observe the selected value displayed in the dropdown
5. Repeat the test with **Price: High to Low**

## Expected Result
- After selecting **Price: Low to High**, that option remains visible as the selected value in the dropdown
- After selecting **Price: High to Low**, that option remains visible as the selected value in the dropdown
- The dropdown accurately reflects the user’s active sort choice

## Priority
Medium

## Type
UI / Functional
