# TC-005: Verify Sort By dropdown includes price sorting options

## Test Case ID
TC-005

## Title
Verify the Sort By dropdown includes both price sorting options

## Related User Story
Sort product listings by price

## Preconditions
- User is on the Product Listing Page (PLP)
- Sort By dropdown is visible

## Test Steps
1. Navigate to the Product Listing Page
2. Open the **Sort By** dropdown
3. Review the list of available sorting options

## Expected Result
- The dropdown includes **Price: Low to High**
- The dropdown includes **Price: High to Low**
- Both options are selectable

## Priority
Medium

## Type
UI / Functional
