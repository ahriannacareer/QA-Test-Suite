# TC-013: Verify user can log in with valid credentials

## Test Case ID
TC-013

## Title
Verify a registered user can log in using valid email and password

## Related User Story
Log into account using valid credentials

## Preconditions
- User is on the Login page
- A registered user account with valid credentials exists
- Email and Password fields are visible
- Login button is available

## Test Steps
1. Navigate to the Login page
2. Enter a valid email address in the Email field
3. Enter the correct password in the Password field
4. Click the Login button

## Expected Result
- Login is successful
- The user is authenticated
- The user is redirected away from the Login page and lands on account page

## Priority
High

## Type
Functional
